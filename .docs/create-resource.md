# Creating Resources

This document provides commands to create various resources in the project using the framework CLI.

## Component

```
yarn framework:create component AppTestComponent
```

## Constant

```
yarn framework:create constant testConstant
```

## Hook Form

```
yarn framework:create form Test
```

## Custom Hook

```
yarn framework:create hook useTest
```

## Layout

```
yarn framework:create layout AppTestLayout
```

## Mutation

```
yarn framework:create mutation CreateTest
```

## Provider

```
yarn framework:create provider AppTestProvider
```

## Query

```
yarn framework:create query GetTestById
```

## Route

```
yarn framework:create route test/index.tsx
```

## Store

```
yarn framework:create store Test
```

## Type

```
yarn framework:create type test
```

## Utils

```
yarn framework:create util test
```

## View

```
yarn framework:create view AppTestView
```
